# Todo list app
<br>

## cloning project

```
git clone https://github.com/Souhardyagayen99/todo-app-using-node-js
```

## install dependencies

```
npm install
```

## run server

```
npm run gayen
```

<p>or</p>

```
npm run start

```
<p>or</p>

```
npm start
```

<p>or</p>

```
nodemon app.js
```

## open this server at your browser

```
http://localhost:8080
```
