# Todo list app
<br>

## cloning project

```
git clone https://github.com/aphrodeosubarno/node-js-todo-app.git
```

## install dependencies

```
npm install
```

## run server

```
npm run gayen
```

<p>or</p>

```
npm run start

```
<p>or</p>

```
npm start
```

<p>or</p>

```
nodemon app.js
```

## open this server at your browser

```
http://localhost:3000
```
